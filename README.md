# 🔤 진짜 쉬운 파닉스 (Phonics Golden Sample)

> 초등 4학년~중학 3학년 학생을 위한 영어 파닉스 학습 + EGU 어휘 학습 웹앱
> 「진짜 쉬운 파닉스는 이렇게 배웁니다」 + 「EGU 영단어 & 품사」 책의 학습법을 디지털로 옮긴 골든샘플

## ✨ 학습 방식

알파벳 소리를 한 글자씩 분리해서 듣고 → 빠르게 합쳐서 단어로 읽는 방식.

```
ham = h /흐/ + a /애/ + m /음/
    = /흐애음/
    → 빠르게 읽어서 /햄/
```

## 📚 커리큘럼 (v9 · 골든샘플)

### 파닉스 본 학습 — 184 단어
- **Level 1: 단모음** (a / e / i / o / u) — 50 단어
- **Level 2: 매직 e** (a_e / e_e / i_e / o_e / u_e) — 24 단어
- **Level 3: 이중자음** (sh / ch / th / ph / wh / ng) — 30 단어
- **Level 4: 이중모음** (ai_ay / ee / ea / oa / ow / oi_oy) — 30 단어
- **Level 5: 사이트워드** (1~5군) — 50 단어

### 짧은 독해 — 12 지문 (Module C)
- Level 1 단모음 5개 / Level 2 매직 e 2개 / Level 3 이중자음 2개 / Level 4 이중모음 2개 / Level 5 사이트워드 1개
- 전체 듣기 / 문장별 듣기 / 한글 토글 / "다 읽었어요" 진도

### EGU 어휘 — 55 Unit × ~20단어 = **1,102 단어** (Module D · v9에서 대폭 확장)
- 「EGU 영단어 & 품사」 책 Unit 01~55 전체 수록
- Unit별 발음 듣기 + 학습 완료 체크
- 빠른 단원 선택 (1~10, 11~20, ... 51~55)
- **단어별 일러스트** (이모지/SVG 자동 매핑, 1100+ 단어)

### 📝 단어 시험지 인쇄 (Module G · v9 신규)
- 단원 다중 선택 (Unit 01~55 자유 조합)
- 2가지 시험 유형:
  - 🇺🇸 영어 → 🇰🇷 한글 (스펠링 보고 뜻 쓰기)
  - 🇰🇷 한글 → 🇺🇸 영어 (뜻 보고 스펠링 쓰기)
- 시험지 / 정답지 토글 (둘 다 켜면 시험지+정답지 같이 출력)
- 문제 순서 섞기 옵션
- 1단/2단/3단 컬럼 레이아웃
- A4 인쇄 최적화 (호매실캠퍼스 ☎ 031-292-2828 헤더 포함)

## 🛠️ 사용 방법

별도 빌드 없음. `index.html`을 브라우저로 더블클릭하면 실행됩니다.
온라인: <https://phonics-golden-sample.vercel.app/>

## 🔧 기술 스택

- React 18 (CDN)
- Tailwind CSS (CDN)
- Babel Standalone (JSX 트랜스파일)
- Web Speech API (TTS)
- localStorage (진도 저장)
- `window.print()` + `@media print` CSS (시험지 인쇄)
- 자체 인라인 SVG 일러스트 약 150개 + 이모지 폴백
